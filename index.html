Description of the Course

The purpose of this course is to provide an introduction to physics, with focus on concepts,
(while using Elementary Algebra) as a tool to guide in the understanding of everyday life.
Will cover Newtonian Mechanics and Electricity & Magnetism.

Make-Up Exams Policy

All tests must be taken on the scheduled date.
NO Make-up exams will be given.
Exception: if student has unplanned medical emergency, a make-up exam may be given
in which case a doctors note will need to be provided or the doctor can contact me directly via email.
Make sure you do not have any schedule conflict.
  • If the first exams is missed, it will be dropped and its weight will
    be counted towards your second exam score for a total weight of 50%.

  • If a second exam is missed, it will result in FAILURE.

  • If the final exam is missed, it will result in FAILURE.
